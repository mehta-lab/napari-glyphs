# glyphs

----------------------------------

## Installation

You can install `napari-glyphs` via [pip]:

    conda create -y -n napari-glyphs python
    conda activate napari-glyphs
    git clone XX 
    cd XX
    pip install -e .
    napari -w napari-glyphs
